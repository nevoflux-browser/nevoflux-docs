Contributions are welcome — code, docs, translations, and bug reports.

## The repositories

- **Browser** — [github.com/nevoflux-browser/nevoflux](https://github.com/nevoflux-browser/nevoflux)
  (Firefox-based; the sidebar, `nevoflux://` pages, and Canvas).
- **Agent** — [github.com/nevoflux-browser/nevoflux-agent](https://github.com/nevoflux-browser/nevoflux-agent)
  (the Rust daemon, providers, skills, tools, and knowledge base).

## Ways to help

- File issues and feature requests on GitHub.
- Improve these docs — they are written in MDX (English `*.mdx`, Simplified Chinese
  `*.zh.mdx`).
- Add or improve **skills** — see [Authoring skills](/en/docs/skills/authoring).
- Help translate the docs and UI into more languages.

## Building from source

- The **browser** builds with Node.js 22+, npm, and Python 3.
- The **agent** is a Rust workspace (toolchain 1.91.0); on Linux it needs the X11 and
  Wayland development libraries.

See each repository's README for full build instructions.
