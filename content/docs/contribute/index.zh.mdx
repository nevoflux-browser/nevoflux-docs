欢迎各种形式的贡献 —— 代码、文档、翻译以及缺陷报告。

## 代码仓库

- **浏览器** —— [github.com/nevoflux-browser/nevoflux](https://github.com/nevoflux-browser/nevoflux)
  (基于 Firefox;侧边栏、`nevoflux://` 页面与 Canvas)。
- **智能体** —— [github.com/nevoflux-browser/nevoflux-agent](https://github.com/nevoflux-browser/nevoflux-agent)
  (Rust 守护进程、提供商、技能、工具与知识库)。

## 参与方式

- 在 GitHub 上提交 issue 和功能请求。
- 改进本文档 —— 使用 MDX 编写(英文 `*.mdx`,简体中文 `*.zh.mdx`)。
- 新增或改进**技能** —— 见[编写技能](/zh/docs/skills/authoring)。
- 帮助把文档与界面翻译成更多语言。

## 从源码构建

- **浏览器**使用 Node.js 22+、npm 和 Python 3 构建。
- **智能体**是一个 Rust 工作区(工具链 1.91.0);在 Linux 上需要 X11 与 Wayland 开发库。

完整构建说明见各仓库的 README。
