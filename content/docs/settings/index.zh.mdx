## `nevoflux://` 页面

NevoFlux 为其内置页面注册了自定义的 `nevoflux://` URL 协议:

| URL | 页面 |
| --- | --- |
| `nevoflux://home` | 新标签页 / 主页 |
| `nevoflux://settings` | 设置(本页) |
| `nevoflux://canvas/{id}` | 打开某个 [Canvas](/zh/docs/canvas) 制品 |
| `nevoflux://import/{share_id}` | 导入分享的 Canvas |
| `nevoflux://history` | 对话历史 |
| `nevoflux://plan/{id}` | 智能体的计划 |
| `nevoflux://render/{job}` | 渲染任务的输出 |
| `nevoflux://brain` | 浏览你的[知识库](/zh/docs/knowledge-base) |

## 设置如何存储

设置界面会写入三处,均位于你的配置目录下:

- **界面偏好**(侧边栏行为、MCP 连接选项等)—— 修改时自动保存。
- **`config.toml`** —— AI Models 标签页在此写入提供商密钥/模型。见
  [配置](/zh/docs/configuration)。
- **Markdown "灵魂" 文件**(`IDENTITY.md`、`SOUL.md`、`USER.md`、`TOOLS.md`、
  `AGENTS.md`)—— General 标签页编辑它们;点击 **Save All Settings** 写入。见
  [General](/zh/docs/settings/general)。

其余标签页(MCP 服务器、Canvas 工具、知识库、[Packs](/zh/docs/settings/packs))会直接
与智能体通信,以管理服务器、工具、知识库和已安装的扩展包。
